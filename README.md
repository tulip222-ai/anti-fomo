# 反 FOMO 🎯

**"这个 AI 名词值得我花时间学吗？"**

用第一性原理快速判断 AI 技术热点，避免盲目跟风。

---

## 🚀 如何使用

### 三步快速开始

1. **📝 输入技术名词**  
   输入你想了解的 AI 技术（如：Mamba、RAG、Attention、Transformer）

2. **🔍 点击分析**  
   系统通过第一性原理进行三维度拆解

3. **📊 查看判决**  
   获得红/黄/绿判决和详细解释

---

## 📊 理解判决结果

### 🟢 绿色 - 投入学习

**含义：** 该技术值得你花时间深入学习

**行动：** 安排时间学习，可能影响你的职业发展

**例子：** 如果你是AI工程师，LLM相关技术通常是绿色

---

### 🟡 黄色 - 持续关注

**含义：** 该技术有前景，但现在不必深入投入

**行动：** 了解基本概念，继续观察发展

**例子：** 一些刚兴起的垂直技术可能是黄色

---

### 🔴 红色 - 暂时观望

**含义：** 该技术还不成熟或适用场景有限，现在投入性价比低

**行动：** 先不投入，继续关注行业动态

**例子：** 某些过度炒作的技术可能是红色

---

## 🧠 第一性原理解释

### 我们如何判断？

反FOMO 使用三个维度进行评估：

#### 1️⃣ **核心价值维度**

- 这项技术解决了什么根本问题？
- 它的核心优势是什么？
- 能否用更简单的方式替代？

#### 2️⃣ **应用前景维度**

- 能应用的场景有多广？
- 有多少开发者会用到它？
- 未来的发展空间有多大？

#### 3️⃣ **学习成本维度**

- 学习难度有多高？
- 需要多少时间掌握？
- 技术依赖和前置知识？

### 判决逻辑

根据这三个维度的综合评估，我们给出判决：

- 如果核心价值强，应用前景广，学习成本合理 → **🟢 绿色**
- 如果某些维度有潜力但还不成熟 → **🟡 黄色**  
- 如果价值不清或应用场景有限 → **🔴 红色**
